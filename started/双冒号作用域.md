# 示例代码

局部：

```c++
#include <vector>
#include <iostream>

int a = 10;

int main() {
  int a = 20;
  std::cout << a << std::endl; // 输出局部的 20
  return 0;
}
```

全局：

```c++
#include <vector>
#include <iostream>

int a = 10;

int main() {
  int a = 20;
  std::cout << ::a << std::endl; // 输出局部的 10
  return 0;
}
```
